# Assets

All media assets for this presentation. Track required assets in `docs/asset-plan.md`.

---

## Directory Structure

```
assets/
├── images/      # Photos, illustrations, icons
├── diagrams/    # SVG files, Mermaid source files
├── videos/      # Video files and demo recordings
└── README.md    # This file
```

## Naming Convention

`[type]-[description]-[version].[ext]`

Examples:
- `diagram-system-architecture-v2.svg`
- `screenshot-claude-artifact-panel.png`
- `video-demo-workflow-generation.mp4`

## Standards

- **Images:** Minimum 2x resolution for projected use
- **Diagrams:** SVG preferred; PNG at 2x minimum as fallback
- **Videos:** MP4, H.264; captions required for all audio content
- **All assets must be bundled locally** — no external URL dependencies in delivery files
