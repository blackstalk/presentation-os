# Slides

Generated presentation files live here. Do not manually edit files in this directory — they are produced by the `html-slides` skill or equivalent tool from content in `docs/`.

---

## Workflow

1. Complete content development (Stages 2–4)
2. Complete design review (Stage 5)
3. Finalize speaker notes (Stage 7)
4. Generate slides using the `html-slides` Claude Code skill
5. Output lands here

## Naming Convention

`[presentation-slug]-v[version].[ext]`

Examples:
- `working-smarter-ai-v1.0.html`
- `mcp-workshop-v2.1.html`

## Output Formats

| Format | Tool |
|--------|------|
| HTML (Reveal.js) | `html-slides` Claude Code skill |
| PDF (export) | Browser print / Reveal.js export |
